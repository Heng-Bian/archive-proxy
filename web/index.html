Archive-proxy

An archive proxy written in Go language supporting zip, tar, 7z, rar(including rar5).

For more information https://github.com/Heng-Bian/archive-proxy/blob/main/README.md

Read a remote file

  1. Paste a URL
  2. Select the encoding of names
  3. List the remote archive files
  4. Select multiple files
  5. Download the selected files
